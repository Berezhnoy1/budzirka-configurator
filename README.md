# BUDZIRKA · Конфігуратор виробів

Веб-конструктор для розрахунку вартості **сантехнічних кабін** і **стільниць Fundermax**.
Замінює поточний AppSheet-застосунок «CALCULATOR BUDZIRKA». Призначений для вбудовування на сайт
[budzirka.com.ua](https://budzirka.com.ua/).

## 🔗 Онлайн (live)

**https://budzirka-configurator.vercel.app**

## Запуск

Відкрий `index.html` подвійним кліком — усе працює без збірки й інтернету (самодостатній файл).

## Деплой / оновлення

Деплой на Vercel (акаунт `mercurial17717-1480`) з ASCII-копії `C:\Users\Mercu\budzirka-configurator`:

```
cp index.html  C:\Users\Mercu\budzirka-configurator\index.html
vercel deploy "C:/Users/Mercu/budzirka-configurator" --prod --yes
```
(Кирилична назва основної теки не годиться для імені Vercel-проєкту, тому деплоїмо з копії.)

## Що вже є (Фаза 1 — прототип)

- 🚻 **Конструктор санкабін**: 8 типів, кількість секцій × ширина × висота, матеріал, фурнітура, лід.
- 🍽️ **Конструктор стільниць**: форми I / Г / П, торці №1–9, вирізи, матеріал, лід.
- 📐 **Живий SVG-ескіз**, що оновлюється під час вводу.
- 🎨 **Свотчі декорів** зі складу (демо-знімок), фільтр за напрямком, бейдж наявності.
- 💰 **Прозорий розрахунок** у реальному часі + цінова категорія клієнта (Прайс 1–5).
- 🧾 **Комерційна пропозиція (КП)** з друком / збереженням у PDF.

## Структура коду (`index.html`, один файл)

Усе в `<script>`, поділено на блоки:

| Блок | Призначення |
|---|---|
| `CONFIG` | **Уся цінова логіка** — базова ставка 170 $/пог.м, надбавки, тарифи торців, знижки. Калібрувати тут. |
| `DECORS` | Статичний знімок матеріалів (код, декор, ядро, поверхня, бренд, наявність, ціна м²). |
| `CABIN_TYPES`, `SHAPES`, `EDGES` | Довідники типів/форм/торців. |
| `state` | Поточний стан конфігуратора. |
| `priceCabins` / `priceCounters` | Формули ціни. |
| `sketchCabins` / `sketchCounter` | Генерація SVG-ескізів. |
| `render*` / `viewWizard` / `*Step` | Рендер екранів. |

## Калібрування цін

Усі цифри-заглушки — у `CONFIG` зверху скрипта:
- `cabins.basePerMeter` — **170** (відоме значення).
- `cabins.decorTier` — надбавка за клас декору.
- `cabins.fittings` — надбавка/знижка за фурнітуру.
- `counters.edgePerMeter` — обробка торця за пог.м.
- `counters.cutoutFee` — ціна вирізу.
- `priceCat` — знижки за ціновими категоріями (1С-прайси).
