# BUDZIRKA Конфігуратор — як працює логіка

Документ пояснює, **як конфігуратор рахує ціну** і з чого складається. Без коду, простими словами.
Демо: **https://budzirka-configurator.vercel.app**

---

## 1. Загальна схема роботи

```
Вибір виробу → Налаштування (кроки) → «Додати до кошторису»
   → Кошторис (кілька позицій + контакти клієнта) → Комерційна пропозиція (PDF)
```

Два напрямки:
- **Сантехнічні кабіни** — ціна за погонний метр перегородки.
- **Стільниці Fundermax** — ціна за площу + обробка торця + вирізи.

Усі дані (матеріали, наявність) — поки **демонстраційний знімок**; реальні підтягнемо зі складу (1С).
Що **вже реальне:** базова ставка кабін **170 $/пог.м**. Решта ставок — **попередні, калібруються**.

---

## 2. Сантехнічні кабіни — формула

```
довжина (м)   = кількість секцій × ширина секції (мм) ÷ 1000
База          = 170 $ × довжина
+ Декор       = База × надбавка за клас декору
+ Фурнітура   = ставка фурнітури ($/пог.м) × довжина
+ Висота      = База × 6% за кожні +500 мм понад 2000 мм
─────────────────────────────────────────────
Вартість позиції = База + Декор + Фурнітура + Висота
```

**Надбавка за клас декору:** стандарт ×1.00 · дерево ×1.08 · преміум ×1.15
**Фурнітура:** нержавіюча сталь 0 · алюміній −15 · нейлон −25 · антивандальна +25 ($/пог.м)

**Приклад:** MAX Standart, 3 секції × 900 мм, висота 2000, декор 0085 (стандарт), нержавіюча сталь
- довжина = 3 × 900 ÷ 1000 = **2.7 м**
- База = 170 × 2.7 = **459 $**
- надбавок немає → **Вартість позиції = 459 $**

---

## 3. Стільниці Fundermax — формула

```
Площа (м²)   = сума по сегментах (довжина × глибина ÷ 1 000 000)
Матеріал     = Площа × ціна матеріалу за м²
+ Торець     = довжина торця (м) × ставка профілю ($/пог.м)
+ Вирізи     = кількість × 15 $
─────────────────────────────────────────────
Вартість позиції = Матеріал + Торець + Вирізи
```

**Форми:** I-подібна (1 сегмент) · Г-подібна (2) · П-подібна (3).
**Ставка торця за складністю:** прямий рез 8 · фаска 12 · профіль/радіус 18 ($/пог.м).
**Ціна матеріалу за м²** береться з картки декору (демо: 130–160 $).

**Приклад:** I-подібна 2000×600, торець №1 (прямий), 1 виріз, декор 0599 Orca (160 $/м²)
- Площа = 2000 × 600 ÷ 1 000 000 = **1.2 м²**
- Матеріал = 1.2 × 160 = **192 $**
- Торець = 2.0 м × 8 = **16 $**
- Вирізи = 1 × 15 = **15 $**
- **Вартість позиції = 223 $**

> Розкрій листів у форматах (4100×640, 2040×1300…) з мінімумом відходу — наступна фаза.

## 4. Цінова категорія клієнта (Прайс 1–5)

До **підсумку кошторису** застосовується коефіцієнт категорії клієнта:

| Категорія | Коефіцієнт |
|---|---|
| Прайс 1 | ×1.00 |
| Прайс 2 | ×0.97 |
| Прайс 3 | ×0.94 |
| Прайс 4 | ×0.91 |
| Прайс 5 | ×0.88 |

```
Разом = (сума всіх позицій × кількість) × коефіцієнт категорії
```
*(Це демо-знижки. У продакшні — реальні прайси з 1С.)*

## 5. Дані, якими оперує конфігуратор

- **Декори** — код, назва, колір, напрямок (кабіни/стільниці), колір ядра, поверхня, бренд (FUNDERMAX/SMART), клас (стандарт/дерево/преміум), **наявність на складі**, ціна за м².
- **Типи кабін** — 8 шт. · **Форми стільниць** — I/Г/П · **Торці** — №1–9.
- **Цінові категорії** — Прайс 1–5.
- **Контакти клієнта** — ім'я, прізвище, компанія, телефон, email, менеджер, коментар.

---

## 6. Де змінюються цифри (для технічного спеца)

Уся цінова логіка зібрана в **одному блоці `CONFIG`** на початку `index.html`:
базова ставка, надбавки за декор/фурнітуру/висоту, ставки торців, ціна вирізу, коефіцієнти прайс-категорій.
Змінити ціну = змінити одне число в `CONFIG`, без переписування коду.

**Що калібруємо разом із замовником:** усі ставки, крім базової 170 $/пог.м (вона вже реальна).
